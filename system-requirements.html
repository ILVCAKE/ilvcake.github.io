System Requirements

To ensure the optimal performance of ReplicaISA, please make sure your system meets the following requirements:

Component Minimum Requirements Recommended Requirements
Graphics Card NVIDIA GeForce GTX 1060 or equivalent NVIDIA GeForce GTX 1080 or equivalent
Processor AMD Ryzen 5 2600 or equivalent AMD Ryzen 7 2700X or equivalent
Memory 8GB RAM 16GB RAM
Operating System Windows 10 64-bit Windows 10 64-bit (latest update)

Note: Meeting the minimum requirements will allow you to run ReplicaISA, but for the best experience, we highly recommend meeting the recommended requirements.
