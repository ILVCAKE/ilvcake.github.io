4. Run as Administrator

ReplicaISA may need administrator privileges to function correctly. üîë

  • Right-click on the ReplicaISA icon and select "Run as administrator."

5. License Authentication Errors

  • "ERROR: AAL IS NOT RUNNING"
    • Make sure ReplicaISA.exe is running and you have pressed '1' to start it. ReplicaISA won't work without ReplicaISA active.
  • "404 ERROR"
    • This indicates a problem with our servers. Please create a support ticket on our Discord server.
  • "HWID NOT MATCHED"
    • This means your HWID isn't recognized. Please create a support ticket on our Discord server.
